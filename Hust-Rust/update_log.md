# Hust-Rust 更新日志

--------------------
**日期**: 2026-05-24
**版本**: v0.1.6+20260524

### 修复内容

1. **for 循环转换优化**
   - 简单条件如 `i < n` 转换为 Rust 原生 `for i in 0..n`
   - 复杂条件如 `(i*i <= n)` 转换为 while 循环

2. **修复 while 循环递增 BUG**
   - 问题：更新语句 `i = i + 1` 未添加到 while 循环体
   - 解决：将更新语句插入 while 循环体的开头

3. **修复转换顺序问题**
   - 问题：`for (i = 2; i*i <= n; i++)` 转换为 `for ((i16 i) = 2; ...)`
   - 原因：变量声明转换在 for 循环转换之后执行
   - 解决：调整转换顺序，for 循环转换优先于变量声明

4. **修复数组声明双分号 BUG**
   - 问题：`i16[4] numbers = {1,2,3,4};` 转换后产生 `;;`
   - 原因：正则表达式未匹配末尾分号
   - 解决：正则改为 `([^}]+)\};` 确保匹配完整声明

### 测试结果
- 质数判断测试通过：2 is true, 17 is true, 25 is false, 97 is true
